# Criterios de Aceptación (Formato Gherkin) — English Master System (EMS)

## 1. Plan personalizado de aprendizaje

**Escenario:** Generación del plan individualizado
**Dado** que un usuario se registra en la plataforma,
**Cuando** ingresa su nivel actual, objetivo (C2) y disponibilidad diaria,
**Entonces** el sistema genera un plan de estudio personalizado basado en los tres métodos científicos validados.

## 2. Sugerencia de tareas auténticas

**Escenario:** Recomendación de tarea semanal
**Dado** que el usuario inicia su sesión semanal,
**Cuando** el sistema identifica su progreso,
**Entonces** sugiere una nueva tarea auténtica (Task-Based Learning) relacionada con su profesión o intereses.

## 3. Análisis de pronunciación

**Escenario:** Evaluación automática del speaking
**Dado** que el usuario graba un ejercicio oral,
**Cuando** el sistema procesa el audio con reconocimiento de voz,
**Entonces** genera un reporte con puntuación de pronunciación, fluidez y coherencia, y ofrece retroalimentación inmediata.

## 4. Métricas de progreso

**Escenario:** Cálculo semanal de progreso
**Dado** que el usuario completa sus actividades diarias,
**Cuando** termina la semana,
**Entonces** el sistema presenta métricas automáticas en vocabulario, gramática, listening y speaking.

## 5. Retroalimentación personalizada del tutor virtual

**Escenario:** Generación de feedback adaptativo
**Dado** que el usuario finaliza una tarea,
**Cuando** el sistema evalúa desempeño mediante NLP y análisis de errores,
**Entonces** muestra recomendaciones específicas para mejorar los aspectos con menor puntuación.

## 6. Gestión de usuarios

**Escenario:** Control de roles y acceso
**Dado** que un administrador accede al panel,
**Cuando** visualiza la lista de usuarios,
**Entonces** puede asignar roles (aprendiz, tutor IA, investigador) y modificar permisos de acceso.

## 7. Visualización de estadísticas globales

**Escenario:** Validación del método mediante datos agregados
**Dado** que un administrador o investigador accede al módulo de reportes,
**Cuando** consulta el panel de rendimiento global,
**Entonces** el sistema muestra resultados agregados y métricas comparativas de eficacia del método sin exponer datos personales.

## 8. Exportación de reportes

**Escenario:** Generación de evidencia científica
**Dado** que un investigador solicita un reporte,
**Cuando** selecciona el formato (PDF/CSV),
**Entonces** el sistema genera el documento con datos anonimizados y listos para análisis externo.
